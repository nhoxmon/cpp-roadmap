Roadmap: Từ C++ tới Software Engineer

Giai đoạn 1: C++ Cơ bản

  • Biến, kiểu dữ liệu, toán tử
  • Câu điều kiện, vòng lặp
  • Mảng 1 chiều & 2 chiều, hàm
  • Con trỏ, cấp phát động
  • Đệ quy, struct, class, File I/O

Giai đoạn 2: OOP & Clean Code

  • Class, Object, Kế thừa, Đa hình
  • Clean Code: DRY, KISS, YAGNI

Giai đoạn 3: Git & GitHub

  • Git cơ bản: commit, push, pull
  • GitHub: fork, pull request, teamwork

Giai đoạn 4: Giải thuật & CTDL

  • Array, Linked List, Stack, Queue
  • Hash Map, Set, Tree, Graph, DP

Giai đoạn 5: Project & Kiến trúc phần mềm

  • Học backend: Python, NodeJS...
  • Design pattern, DevOps cơ bản

Tips học hiệu quả

  • Pomodoro, ghi chép, chia sẻ lại
  • Join cộng đồng: Discord, Reddit, FB Group
